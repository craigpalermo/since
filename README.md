# since
Calculate number of days since a past date.

## Installation
```
npm install -g @unscsprt/since
```

## Usage
```
since "<input>"
```

Supported formats for `<input>` include:

- `MM-DD-YYYY`
- `DD-MM-YYYY`
- `YYYY-MM-DD`

Inputs can either be in the past or future.
